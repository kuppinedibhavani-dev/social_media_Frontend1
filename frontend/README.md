Social Media Content Scheduler (Full Stack Project)

📖 Project Description

This application allows users to schedule, organize, and manage content across multiple social media platforms. Users can create posts, set publish dates, store media URLs, and manage their content in an optimized way. The project is built using a full-stack architecture integrating React (Frontend), Node/Express (Backend), and Supabase (Database).
    
    🔗 Live Deployment
          Netlify URL👉https://socialmediafrontend1.vercel.app/

    

🟦 1. Project Overview

The frontend provides an intuitive and modern interface for users to:
      🔐 Sign Up / Log In using Supabase Auth
      📝 Create, edit, and delete scheduled posts
      📤 Upload images for posts
      📅 View a dashboard of scheduled posts
      🔄 Experience real-time data updates
      📱 Navigate through a fully responsive UI
   It connects seamlessly with the backend REST API and uses Supabase for authentication and storage.


🛠 Tech Stack Used
